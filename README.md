# Umamibot
Telegram bot to track manga chapter releases.

# Built with:
- Node.js
- Telegraf
- Cheerio

# Supported Manga Sources:
- https://mangas.in/
- https://manhwa18.com/
- https://fanfox.net/
- https://lectortmo.com/
- https://www.anzmangas.com/

# Config

Add a .env file with the following variable:
BOT_TOKEN=[your telegram bot token here]
