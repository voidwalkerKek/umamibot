# Umamibot
Telegram bot to track manga chapter releases.

Made with:
- Node.js
- Telegraf
- Cheerio

Supported Sources:
- https://mangas.in/
- https://manhwa18.com/
- https://fanfox.net/
- https://lectortmo.com/
- https://anzmanga.net/

#Usage

Add a .env file with the following variable:
BOT_TOKEN=[your telegram bot token here]
